# 🚀 POSCO - Fullstack Web & Mobile Application

## 📌 Overview

POSCO adalah aplikasi fullstack yang terdiri dari Web, Mobile, dan Backend API dengan sistem terintegrasi.

---

## 🧱 Tech Stack

### 🌐 Web

* React.js / Next.js

### 📱 Mobile

* Flutter / React Native

### ⚙️ Backend

* Node.js (Express.js) / Laravel

### 🗄️ Database

* PostgreSQL

### 🔗 API

* REST API (JSON)

### 🔐 Authentication

* JWT / Laravel Sanctum

### ☁️ Deployment

* VPS / AWS / GCP / DigitalOcean

### 🧪 Testing

* Postman

### 🎨 Design

* Figma

### 🗃️ Version Control

* Git & GitHub

## 👥 Git Workflow

### Branch

* main
* frontend-web
* frontend-mobile
* backend-api

1. Clone repository
2. Buat branch sesuai tugas
3. Coding
4. Push ke branch
5. Pull Request ke main
6. Review & Merge

## ⚙️ Setup

### Clone

```
git clone https://github.com/Kurnia2810/projek-posco.git
cd projek-posco
```

---

### Web

```
cd web
npm install
npm run dev
```

---

### Backend

```
cd backend
npm install
npm run dev
```

---

### Mobile

#### Flutter

```
cd mobile
flutter pub get
flutter run
```

#### React Native

```
cd mobile
npm install
npx react-native run-android
```

---

## 🔐 Environment

Buat file `.env` di backend:

```
PORT=3000
DB_HOST=localhost
DB_USER=postgres
DB_PASS=yourpassword
JWT_SECRET=yoursecretkey
```
